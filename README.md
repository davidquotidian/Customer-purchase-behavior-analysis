# 📊 Analýza nákupného správania zákazníkov

## 📌 O projekte

Projekt sa zameriava na analýzu nákupného správania zákazníkov významného slovenského predajcu elektroniky. Cieľom bolo identifikovať faktory ovplyvňujúce využívanie zliav, vernostného programu a nákupné správanie jednotlivých zákazníckych segmentov.

Analýza vychádza z viac ako 100 000 transakcií a bola realizovaná pomocou PostgreSQL, Pythonu a R.

## 🛠️ Použité technológie

- PostgreSQL
- Python
- R

## 🔍 Metodika

Projekt bol realizovaný podľa nasledovného postupu:

1. Vytvorenie hypotéz
2. Čistenie a príprava dát
3. Spracovanie dát
4. Analýza
5. Interpretácia výsledkov
6. Formulácia odporúčaní pre marketing a podporu predaja

## 📈 Skúmané oblasti

- Vernostný program
- Miera využívania zliav
- Vekové segmenty zákazníkov
- Newsletterový súhlas
- Regionálne rozdiely
- Frekvencia nákupov
- Burn rate vernostných bodov
- Sezónnosť predaja

## 💡 Hlavné zistenia

Analýza ukázala, že najhodnotnejší a najlojálnejší zákazníci zároveň patria medzi najaktívnejších používateľov zliav a vernostných benefitov. Výsledky tiež poukázali na regionálne rozdiely, vekové špecifiká a sezónne vzorce v nákupnom správaní zákazníkov.

## ⚠️ Upozornenie

Projekt predstavuje verejne zdieľateľnú verziu analýzy. Z dôvodu dohody o mlčanlivosti (NDA) boli všetky obchodne citlivé údaje anonymizované alebo randomizované. Použitá metodika, analytický postup a štruktúra riešenia zodpovedajú pôvodnému projektu.
